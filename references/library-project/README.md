# Library Project Reference

This directory contains an older library-project sample kept only as reference material.
It is not current Qiwen Online Judge application source, and it should not be used when
judging the active frontend/backend architecture.

Rules:

- do not import code from this directory into `frontend/`, `backend/`, `judger/`, or `judge-protocol-scala/`
- do not commit generated output under this directory
- prefer searching active source roots directly; root-level ripgrep ignores `references/**`
- if a reference pattern is copied into active source, adapt it to the current domain-first layout first
